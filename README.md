# 🌐 IP Monitoring System

A clean and modular Flask-based web application to analyze IP addresses.

## ✨ Features
- IP Geolocation lookup
- ISP detection
- Modular backend architecture
- Clean UI

## 🧱 Architecture
- Flask (Blueprint-based structure)
- Service layer separation
